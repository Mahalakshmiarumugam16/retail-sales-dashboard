# Retail Sales Analysis Dashboard

## 📌 Overview
This project analyzes retail transaction data to understand sales performance, customer behavior, and profitability.

## 🎯 Objective
To identify key factors affecting sales and profit using data visualization.

## 🛠 Tools Used
- Power BI
- CSV Dataset
- Data Modeling
- DAX

## 🔍 Data Enhancement
The original dataset lacked key business metrics.

So I created:
- Profit (using assumed margin logic)
- Discount (for pricing analysis)
- Region (for geographic segmentation)
- Customer Segment (based on age groups)

## 📊 Key Metrics
- Total Sales: 456K  
- Total Profit: 151K  
- Profit Margin: 33%  
- Customers: 1000  

## 📈 Insights
- Some categories generate high sales but lower profit margins
- Discounts impact profitability significantly
- Customer segments show different purchasing behaviors
